
# Sam Task Manager (Grouped · DnD · Filters · Reminders)

Vite + React project with your grouped task manager.

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
- Push this repo to GitHub (main branch)
- On Vercel: **New Project** → **Import Git Repository** → select this repo → **Deploy**
